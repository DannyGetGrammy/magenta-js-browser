Welcome to Magenta.js

This application allows you to explore machine learning models for music generation using Google's Magenta.js library. You'll be able to generate melodies, create variations, and understand how machine learning can be applied to music creation.

Getting Started

The models load in the background. Once they're ready (indicated by green status indicators at the top), you can start exploring!

Choose one of the tabs above to try different generation approaches:

  • Continuation (MusicRNN): Generate a continuation of a musical sequence using a Recurrent Neural Network
  • Sampling (MusicVAE): Create brand new melodies from a Variational Autoencoder's latent space
  • Interpolation (MusicVAE): Blend between two different melodies to create smooth transitions

About Magenta.js

Magenta.js is the JavaScript implementation of Google's Magenta project, which uses machine learning to create music and art. It's the technology behind tools like Magenta Studio, but gives you direct access to the underlying models.

Machine Learning Concepts

This app demonstrates several key machine learning concepts:

  • Recurrent Neural Networks (RNN): Models that can generate sequential data (like melodies) based on what came before
  • Variational Autoencoders (VAE): Models that learn a compressed representation (latent space) of musical ideas
  • Temperature: A parameter that controls randomness in generation - lower values produce more predictable output, higher values more surprising results
  • Latent Space: A mathematical representation of musical features that the model learns

How MusicRNN Works

MusicRNN is a recurrent neural network that's been trained on a large dataset of melodies. Given a seed melody, it predicts what notes are likely to come next.

Try experimenting with different seed melodies and temperature values to see how they affect the generated continuation.

Understanding MusicVAE

MusicVAE uses a variational autoencoder that compresses musical ideas into a latent space. By sampling from this space, the model can generate completely new melodies.

Higher temperature values will give more varied and potentially interesting results, but might be less musically coherent.

How Interpolation Works

Interpolation works by encoding two different melodies into the latent space, then creating a series of points between them. These points are then decoded back into melodies.

This demonstrates how the latent space that MusicVAE learns represents musical ideas in a continuous way, allowing for smooth transitions between very different melodies.
